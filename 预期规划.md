# StickyDesk 预期规划（v1.0.1 之后）

## 已完成（1.0.1 前后）

- Tauri 架构迁移与主功能对齐
- 系统托盘：关闭隐藏、托盘恢复、菜单退出
- 静默开机自启
- 单实例保护（重复启动聚焦已有窗口）
- Future Task 完成/撤销、删除二次确认、展开编辑

## 下一阶段（优先级从高到低）

1. 点击穿透模式（带锁定开关）
2. 边缘自动收起与悬停唤醒
3. 数据导出（JSON）与导入基础能力
4. 全局快捷键（显示/隐藏窗口、快速新建）
5. 多窗口拆分（便签/任务独立浮窗）

## 中长期探索

- 云同步与多端协作
- 团队协作玩法（例如 active 时间排行榜）
- 更细粒度的性能优化（列表虚拟化、懒加载）
