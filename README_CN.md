# StickyDesk

[English](./README.md)

一个轻量级桌面便签应用，基于 **Electron + React** 构建（优先支持 Windows）。  
功能包括便签标签、搜索、置顶、任务规划（含重要性颜色）、可选悬浮组件，以及可选的活跃时长统计。

> 状态：早期开发阶段 / MVP 进行中。

---

## 功能

### 便签

- 新建 / 编辑 / 删除便签
- 置顶便签
- 多标签（Tag）与分类（可选）
- 快速搜索（标题/内容 + 标签过滤）

### 任务（计划）

- 简单任务列表 + 截止时间
- 重要性分级并用颜色深浅表示（例如：浅红 → 深红）

### 界面 / 交互

- 始终置顶的悬浮窗口（可选）
- 便签卡片悬浮高亮 / 轻微缩放
- 右键菜单（计划中）
- 列表内拖拽排序（计划中）

### 可选：活跃时长统计

- 基于系统空闲时间统计活跃时长（不使用全局输入钩子）
- 在悬浮组件中展示活跃时长

---

## 技术栈

- **Electron**（主进程）
- **React**（渲染进程）
- **TypeScript**（推荐）
- **Vite**（推荐）或你选择的打包工具
- 存储：SQLite（可选）/ 基于 JSON 的存储（MVP 更友好）

## 路线图

- [ ] 便签 MVP：CRUD + 标签 + 搜索
- [ ] 任务：重要性颜色 + 截止时间
- [ ] 悬浮组件：始终置顶 + 时长 + 今日摘要
- [ ] 拖拽排序（React DnD）
- [ ] 右键菜单操作
- [ ] 可选：SQLite / FTS 搜索
- [ ] 开机自启动（可选）
